Quasi-invariant in the elastic configuration Let \(d\) be the geometric dimension. If

\[\mathbf{F}_a = (1 - \gamma) \mathbf{f}_0 \otimes \mathbf{f}_0 + \frac{1}{\sqrt{1 - \gamma}} (\mathbf{I} - \mathbf{f}_0 \otimes \mathbf{f}_0)\]

then

\[I_{4f_0}^E = I_{4f_0} \frac{1}{(1+\gamma)^2}\]

If

\[\mathbf{F}_a = (1 + \gamma) \mathbf{f}_0 \otimes \mathbf{f}_0 + \frac{1}{\sqrt{1 + \gamma}} (\mathbf{I} - \mathbf{f}_0 \otimes \mathbf{f}_0)\]

then

\[I_{4f_0}^E = I_{4f_0} \frac{1}{(1+\gamma)^2}\]

pulse.material.holzapfelogden module¶

class pulse.material.holzapfelogden.HolzapfelOgden(activation=None, parameters=None, active_model='active_strain', T_ref=None, eta=0.0, isochoric=True, compressible_model='incompressible', geometry=None, f0=None, s0=None, n0=None, *args, **kwargs)[source]¶

Bases: pulse.material.material_model.Material

Transversally isotropic version of the Holzapfel and Ogden material model

\[\mathcal{W}(I_1, I_{4f_0}) = \frac{a}{2 b} \left( e^{ b (I_1 - 3)} -1 \right) + \frac{a_f}{2 b_f} \left( e^{ b_f (I_{4f_0} - 1)_+^2} -1 \right)\]

where

\[(\cdot)_+ = \max\{x,0\}\]

Reference

[1] Holzapfel, Gerhard A., and Ray W. Ogden. “Constitutive modelling of passive myocardium: a structurally based framework for material characterization. “Philosophical Transactions of the Royal Society of London A: Mathematical, Physical and Engineering Sciences 367.1902 (2009): 3445-3475.

W_1(I_1, diff=0, *args, **kwargs)[source]¶

Isotropic contribution.

If diff = 0, return

\[\frac{a}{2 b} \left( e^{ b (I_1 - 3)} -1 \right)\]

If diff = 1, return

\[\frac{a}{b} e^{ b (I_1 - 3)}\]

If diff = 2, return

\[\frac{a b}{2} e^{ b (I_1 - 3)}\]
W_4(I_4, diff=0, *args, **kwargs)[source]¶

Anisotropic contribution.

If diff = 0, return

\[\frac{a_f}{2 b_f} \left( e^{ b_f (I_{4f_0} - 1)_+^2} -1 \right)\]

If diff = 1, return

\[a_f (I_{4f_0} - 1)_+ e^{ b_f (I_{4f_0} - 1)^2}\]

If diff = 2, return

\[a_f h(I_{4f_0} - 1) (1 + 2b(I_{4f_0} - 1)) e^{ b_f (I_{4f_0} - 1)_+^2}\]

where

\[h(x) = \frac{\mathrm{d}}{\mathrm{d}x} \max\{x,0\}\]

is the Heaviside function.
